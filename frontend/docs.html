📚 Web3 Wallet & ERC20 Token Documentation

Overview

This project provides a complete, production-ready Web3 wallet system with a custom ERC20 token implementation built from scratch using Solidity and ethers.js.

✨ Key Features:
  • Custom ERC20 Token: Full ERC20 implementation with mint, burn, and ownership capabilities
  • Ether Wallet: Manage ETH deposits/withdrawals with secure balance tracking
  • Token Management: Deposit, withdraw, and transfer ERC20 tokens
  • Multi-Token Support: Register and manage multiple ERC20 tokens
  • Web Frontend: Beautiful, responsive UI for wallet interactions
  • Comprehensive Testing: Full test suite with 100+ test cases
  • Production Ready: Hardhat framework with deployment scripts

Quick Start

1. Installation

npm install

2. Compile Contracts

npm run compile

3. Run Tests

npm test

4. Start Local Network

npm run node

In another terminal:

npm run deploy:localhost

5. Run Interactions

npx hardhat run scripts/interact.js --network localhost

Smart Contracts

MyToken.sol - Custom ERC20 Implementation

Function Description Parameters
transfer() Transfer tokens to address to, amount
approve() Approve spender spender, amount
transferFrom() Transfer on behalf from, to, amount
mint() Create new tokens (owner only) to, amount
burn() Destroy tokens amount

EtherWallet.sol - Wallet Management

Function Description Parameters
depositEth() Deposit ETH payable
withdrawEth() Withdraw ETH amount
depositToken() Deposit ERC20 token tokenAddress, amount
withdrawToken() Withdraw ERC20 token tokenAddress, amount
transferToken() Transfer tokens between users tokenAddress, to, amount

Deployment

Local Development (Hardhat)

npm run deploy:localhost

Sepolia Testnet

First, set up your .env file: 
SEPOLIA_RPC_URL=https://sepolia.infura.io/v3/YOUR_KEY
PRIVATE_KEY=your_private_key
ETHERSCAN_API_KEY=your_api_key
npm run deploy:sepolia

Mainnet

npm run deploy -- --network mainnet

Frontend Setup

Update Contract Addresses

Edit frontend/scripts/wallet-app.js and update:

const CONTRACTS = {
  TOKEN_ADDRESS: "0x...", // Your deployed token
  WALLET_ADDRESS: "0x...", // Your deployed wallet
};

Install ethers.js

Add to your index.html:

<script src="https://cdn.ethers.io/lib/ethers-5.0.umd.min.js"></script>

Open in Browser

Simply open frontend/index.html in your browser with MetaMask installed.

const token = await MyToken.deploy("My Custom Token", "MCT", 5000000n);

Interact with Deployed Contract

Use scripts/interact.js as a template for custom interactions.

Security Considerations

  • 🔐 Never commit .env file with private keys
  • ✅ Always test on testnet before mainnet deployment
  • ⚠️ Use reputable RPCs (Infura, Alchemy, QuickNode)
  • 🔍 Consider contract audit before production use
  • 💾 Back up your private keys securely

Troubleshooting

Connection Issues

  • Ensure MetaMask is installed and connected to the correct network
  • Check that contract addresses are correct in wallet-app.js
  • Verify RPC URL in hardhat.config.js

Transaction Failures

  • Check account balance (ETH for gas)
  • Verify token approval before deposits
  • Review contract state and permissions

Compilation Errors

  • Run npm install to install dependencies
  • Clear cache: npm run clean
  • Check Solidity version in hardhat.config.js
